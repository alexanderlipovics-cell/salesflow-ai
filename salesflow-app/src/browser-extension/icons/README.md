# Extension Icons

## Benötigte Icons

Erstelle die folgenden PNG-Icons (oder nutze ein Tool wie https://www.favicon-generator.org/):

- `icon16.png` - 16x16 px
- `icon32.png` - 32x32 px  
- `icon48.png` - 48x48 px
- `icon128.png` - 128x128 px

## Quick-Start ohne Icons

Für Testzwecke kannst du das Manifest anpassen und die Icon-Zeilen entfernen.
Chrome zeigt dann ein Standard-Icon an.

## Empfohlenes Design

- Hintergrund: #667eea (Sales Flow Lila)
- Vordergrund: Weiß
- Symbol: "SF" oder ein Raketen-Icon
